# SmartBudget – Landing Page

## 📖 Descripción
SmartBudget es una landing page creada como práctica de desarrollo web.  
El objetivo es mostrar las características principales de la aplicación de manera clara, ordenada y responsiva.

Este proyecto se realizó paso a paso aplicando buenas prácticas modernas de frontend.

---

## 🛠️ Tecnologías usadas
- **HTML semántico** para una estructura clara.
- **Metodología BEM** para nombrar las clases de forma ordenada.
- **SASS con estructura 7-1** para organizar los estilos en carpetas.
- **Flexbox y CSS Grid** para el layout responsivo.
- **Bootstrap 4** para navbar, cards y botones.
